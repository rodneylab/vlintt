<h1 align="center">
  vlintt
</h1>

Basic CLI tool to lint vtt (Web Video Text Tracks Format or WebVTT) files.

Currently only wraps lines longer than 42 characters.

Example usage:

```shell
vlintt --output text.vtt  text.mp3.vtt
```
